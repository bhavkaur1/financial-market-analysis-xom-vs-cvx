# Risk-Adjusted Stock Analysis: XOM vs CVX

### Overview
This project compares ExxonMobil (XOM) and Chevron (CVX) using historical stock data to evaluate return, volatility, and risk-adjusted performance.  
The goal is to move beyond price charts and assess which stock delivers better returns relative to risk.

### What I Did
- Pulled historical daily stock data from Yahoo Finance using Python
- Calculated daily returns and volatility
- Visualized price trends and return distributions
- Compared mean returns, volatility, and risk-adjusted returns

### Key Insight
Although Chevron appears slightly less volatile visually, ExxonMobil delivers **higher risk-adjusted returns**, indicating stronger return efficiency per unit of risk.

### Tools
Python, Pandas, NumPy, Matplotlib, yfinance, Google Colab
